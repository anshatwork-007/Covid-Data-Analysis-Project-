# Experiment-19 & 20  
## Title: COVID Data Analysis (Project)  

---

## 🎯 Aim  
To analyze COVID-19 data using Python and visualize trends, patterns, and insights through various data analysis and visualization techniques.

---

## 🎯 Objectives  
- To understand real-world dataset analysis  
- To clean and preprocess COVID-19 data  
- To analyze trends such as cases, deaths, and recoveries  
- To visualize data using charts and graphs  
- To derive meaningful insights from pandemic data  

---

## 💻 Software Required  
- Python  
- Jupyter Notebook / Google Colab  
- Libraries: Pandas, NumPy, Matplotlib, Seaborn, Plotly  

---

## 📘 Theory  
COVID-19 data analysis involves studying large datasets related to confirmed cases, recoveries, and deaths across different regions and time periods. Data analysis helps in identifying trends, understanding the spread of the virus, and making informed decisions.

Using Python libraries, data can be cleaned, processed, and visualized effectively. Visualization techniques help in better understanding patterns such as growth rate, peak periods, and regional comparisons.
